# MailGuard AI Ultra — API Reference

Base URL: `https://app.mailguard.ai` (production) · `http://localhost:8000` (local)

All endpoints under `/api/v1`. Auth is `Authorization: Bearer <jwt>` unless noted.
OpenAPI / Swagger UI: `/docs` (auto-generated by FastAPI).

---

## Auth

| Method | Path | Description |
| --- | --- | --- |
| POST | `/auth/register` | Create user. |
| POST | `/auth/login` | Issue access token. |
| GET  | `/auth/me` | Current user. |
| PATCH| `/auth/me` | Update preferences (language, voice, phone). |

```json
POST /auth/login
{ "email": "raj@example.com", "password": "..." }

// → 200
{ "access_token": "...", "token_type": "bearer", "expires_in": 86400 }
```

## Accounts

| Method | Path | Description |
| --- | --- | --- |
| GET | `/accounts` | List connected mailboxes. |
| GET | `/accounts/gmail/start` | Begin Gmail OAuth. |
| GET | `/accounts/gmail/callback` | Exchange code for tokens. |
| GET | `/accounts/outlook/start` | Begin Outlook OAuth. |
| GET | `/accounts/outlook/callback` | Exchange code for tokens. |
| DELETE | `/accounts/{id}` | Disconnect account. |

## Emails

| Method | Path | Description |
| --- | --- | --- |
| GET | `/emails` | List (paginated, filterable). |
| GET | `/emails/{id}` | Full email detail. |
| GET | `/emails/semantic/search?q=...` | RAG semantic search. |

Query params for `/emails`: `page`, `size`, `category`, `priority`, `search`.

```json

```

---

## Rate limits

- Per-user token bucket: 120 requests / 60s burst, refills at 2 rps.
- WhatsApp dispatch: throttled per-user (configurable).

## Versioning

Breaking changes increment the URL prefix (`/api/v2`).
